# FiraCode-hy
Fork of the FiraCode font with added ligatures for the hy programming language

ligature mapping:
```
(fn     -> (λ
(defn   -> (ƒ
(sum    -> (∑
(in     -> (∈
(not-in -> (∉
(and    -> (∧
(or     -> (∨
(not    -> (¬
(for    -> (∀
(None   -> (∅
(some   -> (∃
```

Based on:
--------
https://github.com/tonsky/FiraCode (base font)

https://github.com/hylang/vim-hy   (idea)

Made using FontForge
